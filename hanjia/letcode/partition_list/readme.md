86.分隔链表
给定一个链表和一个特定值x，对链表进行分隔，使得所有小于x的节点都在大于或等于x的节点之前。
你应当保留两个分区中每个节点的初始相对位置。
示例：
  输入：head = 1->4->3->2->5->2, x=3
  输出：1->2->2->4->3->5
